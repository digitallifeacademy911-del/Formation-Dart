## Écrire des commentaires
Lorsqu'on écrit du code en programmation il est important de prendre en compte le code de la scalabité du code. Un commentaire est un marquer qui servira à indiquer le pourquoi, à quoi çà sert et comment cela s'utilise ici. En dart on retrouve 4 façon d'écrire un commentaire :
- Le double slash <code>//</code> pour écrire des commentaires sur un ligne;
- Le slash astérisque <code>/* */</code> pour écrire des commentaires sur plusieurs ligne;
- Le triple slash <code>///</code> pour rédiger la documentation sur une ligne;

## Générer un documentation
> Pour génerer automatiquement une documentation dart possède un package <code>doc</code> qui génère de facon automatique des fichiers html qui présentant les différentes fonctions et code présent dans les fichiers du dossier <code>lib</code>.Pour commencer créer un projet dart en utilisant la commande suivante :

````bash
$ dart create <le_nom_de_votre_projet>
````
